## Comment

한 화면에 명언을 랜덤으로 생성해주는 간단한 앱이지만

처음으로 만들어 본 앱이라 간단한 기능만을 담고있어도 뿌듯했다.

기본으로 만들었던 부분에 더 나은 UI 를 적용해보면 좋을 것 같아서

앱의 컨셉에 대해 생각해보고 아이콘과 색상도 결정하였다.

덕분에 asset 을 다루는 방법에 대해 좀 더 알게 된 것 같다.
